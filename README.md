#  Digital Business Card Generator

A responsive web application that allows users to create a personalized digital business card by entering their details, selecting a template, and downloading the card as a PNG image or PDF.

---

##  Features

- Enter personal details (Name, Job Title, Company, Email, Phone, Website, Location)
- Upload a profile picture
- Add social media links (LinkedIn & GitHub)
- Choose from multiple business card templates
- Generate a professional digital business card
- Download the card as a PNG image
- Download the card as a PDF
- Responsive design for desktop and mobile devices

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- html2canvas
- jsPDF

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Enter your details.
5. Upload a profile image.
6. Select a business card template.
7. Click **Generate Card**.
8. Download the card as a PNG image or PDF.
