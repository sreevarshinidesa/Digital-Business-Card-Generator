# Digital Business Card Generator

A responsive web application that allows users to create personalized digital business cards. Users can enter their professional details, upload a profile picture, choose from multiple templates, and download the generated card as a PNG image or PDF.

## Features

- Create a personalized digital business card
- Upload a profile image
- Add contact information
- Add LinkedIn and GitHub profile links
- Choose from multiple business card templates
- Generate the business card
- Download as PNG
- Download as PDF
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript
- html2canvas
- jsPDF
- Font Awesome

## How to Run

1. Clone the repository.

```
git clone https://github.com/sreevarshinidesa/Digital-Business-Card-Generator.git
```

2. Open the project folder.

3. Open `index.html` in your browser or run it using Live Server.

## Usage

1. Enter your personal and professional details.
2. Upload a profile image.
3. Select a business card template.
4. Click **Generate Card**.
5. Download the business card as a PNG image or PDF.
